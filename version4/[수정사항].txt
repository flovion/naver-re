■ Version 4 수정사항 (2026-03-19)

1. 단지 비교 기능 추가 - 방식1 (단지 비교 탭)
   - 상단에 "단지 조회" / "단지 비교" 탭 메뉴 추가
   - 비교 탭에서 단지 2~3개 검색 후 평형 선택
   - 보기 항목 선택: 실거래가 / 매도호가 / 둘 다
   - "비교 보기" 버튼 클릭 시 나란히 표시
   - 단지 2개 이상 + 평형 선택 완료 시 버튼 활성화

2. 단지 비교 기능 추가 - 방식2 (현재화면 통합)
   - 단지 조회 후 "+ 비교 단지 추가" 버튼 표시
   - 검색창 등장 → 두 번째 단지 선택 가능
   - 비교 단지도 독립적인 평형탭 + 실거래가 + 매도호가 표시
   - "✕ 비교 해제" 버튼으로 제거
   - 파란색 "비교" 배지로 구분

3. 면적 단위 전환 기능 추가 (㎡ ↔ 평)
   - "평형 선택" 옆 "평으로 보기" 버튼으로 전환
   - 평형 탭 + 매도호가 면적 컬럼 동시 변환
   - 단지 조회 탭 / 단지 비교 탭 / 방식2 비교 모두 연동
   - API 재호출 없이 즉시 변환 (data 속성 활용)
   - 변환 공식: ㎡ ÷ 3.3058 = 평

4. 즐겨찾기 접근성 개선 (version 3에서 이월)
   - 헤더 우측 "⭐ 즐겨찾기" 버튼 추가

5. 매도호가 면적 표기 개선 (version 3에서 이월)
   - "중개사 N곳" 배지 두 줄 깨짐 수정
