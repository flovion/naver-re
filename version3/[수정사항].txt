■ Version 3 수정사항 (2026-03-19)

1. 단지 상세정보 강화
   - 세대당주차 표시 (예: 1.53대)
   - 난방 방식 표시 (예: 개별난방)
   - 건설사 표시 (예: 삼성물산(주), GS건설(주))
   - 네이버 /api/complexes/{complexNo} 엔드포인트 추가 호출
   - overview + detail API 병렬 호출로 로딩 속도 유지

2. 즐겨찾기 접근성 개선
   - 헤더 우측에 "⭐ 즐겨찾기" 버튼 추가
   - 버튼 클릭 시 즐겨찾기 섹션으로 스크롤 이동
   - 즐겨찾기 없을 경우 안내 메시지 표시

3. 매도호가 목록 표시 수정
   - "중개사 N곳" 배지가 가격과 같은 줄에 있어 두 줄로 깨지는 문제 수정
   - 배지를 가격 아래 별도 줄(display:block)로 분리하여 깔끔하게 표시

4. 반응형 디자인 (version 2에서 이월)
   - 모바일(600px 이하) 최적화 레이아웃

5. 실거래가 통계 (version 2에서 이월)
   - 평균 / 최고 / 최저 가격 박스 표시
